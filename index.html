CSS Transforms and Transitions

  • Transforms
  • Transitions

Let the CSS fun begin...

A lot of my cheat sheets so far have been dealing with the main building blocks like selectors and positioning, and then moving on to the basics of text and images.

Now that those basics have been covered the fun of CSS can really start by visually manipulating elements using transitions and transforms.

Transforms...

The transform property applies a 2D or 3D transformation to an element.

  • scale()
  • skew()
  • translate()
  • rotate()
  • matrix()
  • perspective()
.element {
transform: translateY(40px);
}

scale()

...affects the size of the element.

  • scale(x,y)
  • scaleX()
  • scaleY()
  • scaleZ(z)
  • scale3d(x,y,z)
.foo {
transform: scaleX(40px);
}
.foo {
transform: scaleY(40px);
}
.foo {
transform: scale(40px, 40px);
}
.foo {
transform: scaleZ(40px);
}
.foo {
transform: scale3D(40px, 40px, 40px);
}

skew()

... tilts an element to the right or left.

  • skewX()
  • skewY()
  • skew(x,y)
.foo {
transform: skewX(10px);
}
.foo {
transform: skewY(10px);
}
.foo {
transform: skew(10px 10px);
}

translate()

...moves an element sideways or up and down.

  • translateX()
  • translateY()
  • translate(x,y)
  • translateZ()
  • translate3d(x, y, z)
.foo {
transform: translateX(20px);
}
.foo {
transform: translateY(20px);
}
.foo {
transform: translate(20px 20px);
}
/*z moves element toward viewer (negative value moves away)*/
.foo {
transform: translateZ(20px);
}
.foo {
transform: translateXYZ(20px 20px 20px);
}

rotate()

...rotates element clockwise from current position.

  • rotate(deg)
  • rotateX()
  • rotateY()
  • rotate3d(x,y,z)
.foo {
transform: rotate(40deg);
}
.foo {
transform: rotateX(40deg);
}
.foo {
transform: rotateY(40deg);
}
.foo {
transform: rotate3d(40deg 40deg 40px);
}

matrix()

...combines all transforms into one.

It is possible to have multiple values on the transform property such as...

.element {
transform: translateY(40px) scale(20);
}

The matrix property combines all transforms into one BUT it is not written by hand and instead use tools to create them.

So this...

.element {
transform: translateY(40px) scale(20);
}

Becomes this...

.element {
transform: matrix(0.7071067811865475, 0.7071067811865476, -0.7071067811865476, 0.7071067811865475, -0.7071067811865497, 34.648232278140824);
}

perspective()

...doesn't affect the element itself but instead affects the 3D transforms of the child elements for depth perspectives.

.element {
transform: perspective 100px, rotate(25deg);
}

transform-style...

...sets whether children of an element are positioned in the 3D space or are flattened in the plane of the element.

.element {
transform-style: preserve-3d;
}
/*otherwise transform-style: flat;*/

Transitions...

CSS transitions allow you to control animation speed and change property values smoothly over a given duration

  • property
  • delay
  • duration
  • timing-function
  • transition
.element {
transform: rotate(25deg);
transition: <property> <duration> <timing-function> <delay>
}

transition-property

...is the property you want to have the effect on.

.foo {
color: #000;
transition: color 2s ease 4s;
}
.foo:hover {
color: #fff;
}

NOTE - "all" can be a property too...but not all elements can be transformed or animated!

transition-delay

...specifies a delay (in seconds) for the transition effect.

.foo {
color: #000;
transition: color 2s ease 4s;
transition-delay: 2s;
}
.foo:hover {
color: #fff;
}

transition-duration

...specifies the duration (in seconds or milliseconds) for the transition effect

.foo {
color: #000;
transition: color 2s ease 4s;
transition-delay: 2s;
transition-duration: 2s;
}

transition-timing-function

...specifies the speed curve of the transition effect.

  • ease(default)
  • linear
  • ease-in
  • ease-out
  • ease-in-out
  • cubic-bezier(n,n,n,n)
.foo {
transform: rotate(40deg);
transition-timing-function: ease-in-out;
}

transition

...is the shorthand for transition-property transition-duration transition-timing-function and transition-delay.

.element {
transition: <property> <duration> <timing-function> <delay>;
}

You can also add multiples seperated by comma:

.element {
transition: width 2s, height 2s, transform 2s;
}
